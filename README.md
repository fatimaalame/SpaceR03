# TR03 -> Graphique 2D interactif
Projet réalisé en JavaScript avec la librairie SVG.js.

## Thème
Le projet représente une scène spatiale interactive avec une fusée, une planète, des étoiles, une météorite et une jauge de carburant hihi.
Du coup, j’ai décidé de réaliser une scène spatiale animée. Le dessin représente un fond spatial avec une fusée, plusieurs planètes, des étoiles et quelques éléments comme des astéroïdes ou un satellite. L’objectif n’est pas de créer un jeu complet, mais une illustration interactive où l’utilisateur peut déclencher différentes animations en cliquant sur les objets.

La fusée peut démarrer lorsqu’on clique dessus: une flamme apparaît et elle se déplace vers le haut. Les planètes réagissent aussi au clic en bougeant légèrement. Lorsqu’une étoile est cliquée, toutes les étoiles scintillent. Un bouton permet également d’activer un mode “hyperspace”, qui modifie l’arrière-plan et donne une impression de déplacement rapide dans l’espace.

Le projet utilise plusieurs composants graphiques SVG simples, comme des cercles, rectangles, polygones, lignes et textes. Les animations principales concernent le scintillement des étoiles et le mouvement léger des planètes.

## Objectif
Créer un contenu graphique 2D animé et interactif sur le web.

## Contraintes du travail
- Au moins 12 composants graphiques
- Au moins 2 animations prédéfinies
- Au moins 4 interactions qui modifient le contenu graphique

## Fichiers
- `index.html` : page web qui charge le projet
- `script.js` : script JavaScript contenant le graphique SVG.js
- `description.pdf` : description du thème et des interactions
